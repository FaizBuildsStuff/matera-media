# Sanity Studio Structure

This document outlines the content structure configured in the Sanity Studio (`sanity/structure.ts`).

## Singleton Documents (Pages)

- **Home Page**
  - Title: Home Page
  - Schema Type: `page`
  - Document ID: `home`

- **Ad Creatives Page**
  - Title: Ad Creatives
  - Schema Type: `servicePage`
  - Document ID: `service-ad-creatives`

- **Organic Content / YouTube Page**
  - Title: Organic Content / YouTube
  - Schema Type: `servicePage`
  - Document ID: `service-organic-content-youtube`

- **SaaS Videos Page**
  - Title: SaaS Videos
  - Schema Type: `servicePage`
  - Document ID: `service-saas-videos`

## Legal

- **Privacy Policy**
  - Title: Privacy Policy
  - Schema Type: `legalPage`
  - Document ID: `legal-privacy-policy`

## Lists (Collections)

- **Inquiries**
  - Schema: `inquiry`
  - Contact/inquiry submissions.

- **Other Documents**
  - Any other document schemas not explicitly listed above will appear at the bottom of the studio sidemenu.
